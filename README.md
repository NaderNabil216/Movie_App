# Movie_App
## Featues
#### Discover Movies
- Movies Listing
- paging
- swipe to refresh
- handle empty state
- handle network state change
#### Movie Details 
- preview movie details
- fetching additional movies official trailers over Youtube

## Architecture 
- Multi Modules
- Clean Architecture
- MVVM

## Tech Stack 
- Retrofit
- Coroutines
- Hilt
- Flow & Livedata
- Navigation Component
